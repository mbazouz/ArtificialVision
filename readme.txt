-----------------------------------------------------------
-----------------ARTFICIAL VISION PROJECT------------------
-----------------------------------------------------------
Aperçu: ARTIFICIAL VISION PROJECT est une application
		conçu pour comparer la semblabilité entre un 
		groupe d'images. 
-----------------------------------------------------------
-----------------------------------------------------------
Langage: 		JAVA
IDE:	 		NETBEANS
Bibliothèques:	- jai_codec.jar
				- jai_core.jar
				- mlibwrapper_jai.jar
				- sun-jai_core.jar
Developpée par: Mohamed Ben Azouz 3éme Embarqué
-----------------------------------------------------------
-----------------------------------------------------------
Exigences: Vous devez avoir JRE installé sur votre PC
		   pour avoir éxecuter cette application.
-----------------------------------------------------------
-----------------------------------------------------------
Execution:
 1 - Placer les images que vous voulez comparer la 
	 semblabilité entre eux dans le dossier "samples"
 2 - Executer le fichier "execute.bat"
 3 - Choisir l'image que vous voulez la comparer avec
	 les autres.
 4 - Une interfaçe ainsi va afficher la semblabilité entre
	 l'image choisi et les autres images.
